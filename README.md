## Project Description
This project is a movie search application built using ReactJS, the TMDB API, Font Awesome, Tailwind CSS, and Axios. The application allows users to search for movies and view details such as the movie's title, release year, rating, and description.
The project is an excellent example for beginners who want to learn about integrating external APIs with React and how to use popular front-end libraries and tools such as Tailwind CSS and Fontawesome.

# Technologies Used
- ReactJS
- TMDB API
- Font Awesome
- Tailwind CSS
- Axios
